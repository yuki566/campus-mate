# CampusMate (キャンパスメイト) 🎓
**金沢工業大学（KIT）生のための、時間割・課題・バイト管理オールインワンアプリ**

![App Icon or Banner](ここにアプリのトップ画面やアイコンの画像を貼る.png)
## 📖 概要
「CampusMate」は、大学生活に必要なスケジュール管理を一つのアプリで完結させるために開発されたPWA（プログレッシブウェブアプリ）です。
時間割の出席管理、課題の提出期限、アルバイトのシフト管理を統合し、学生生活の「うっかり忘れ」を防ぎます。

**デモサイト:** [https://campus-mate-1.netlify.app](https://campus-mate-1.netlify.app)
*(iPhoneの場合はSafariで開き「ホーム画面に追加」することでアプリとして動作します)*

## ✨ 主な機能

### 1. 📅 時間割 & 出席管理
* **KITの授業時間に対応:** 1限〜5限の時間設定をプリセット済み。
* **出席カウンター:** タップ操作で「出席」「欠席」「遅刻」をカウントし、履修状況を可視化。
* **柔軟な編集:** 授業名、教室、担当教員を簡単に登録・編集可能。

### 2. 📝 課題管理 (To-Do)
* **提出期限の可視化:** 締め切り日時を設定し、リストとカレンダーで確認可能。
* **スマートな入力:** デフォルトで「期限の30分前」を開始時間に自動設定する入力補助機能。
* **完了管理:** チェックを入れるとグレーアウト＆取り消し線で達成感を演出。

### 3. 🗓 統合カレンダー (Month / Week / Day)
* **月表示:** 全体の予定を俯瞰。完了した課題は青いドットで表示。
* **週表示:** 授業、課題、バイトの詰まり具合を色帯で直感的に把握（文字情報を排除し視認性を確保）。
* **日表示:** 1日の詳細なタイムラインを表示。

### 4. ⚙️ その他
* **通知設定:** 課題やバイトの開始前にローカル通知を設定可能（※ネイティブ化対応時）。
* **完全オフライン対応:** データは端末内（SharedPreferences）に保存されるため、通信環境がなくても高速に動作。

## 🔥 こだわった点・苦労した点

### 1. タイムゾーンと日付またぎによる「時間のズレ」の解消
**【課題】**
開発当初、予定の時間を変更するドラムロール（TimePicker）を操作すると、内部的な日付計算の基準がずれてしまい、意図しない時間（例: 20:00を選択したのに02:00になる等）が保存されるバグが発生しました。

**【解決策】**
`CupertinoDatePicker` に渡す初期値（`initialDateTime`）の生成ロジックを根本から見直しました。「システム上の現在時刻」と「ユーザーが指定したい時刻」が混在しないよう、常に**「操作当日の年月日 + 設定したい時分」**を厳密に結合して初期化する処理を実装することで、正確な時間保存を実現しました。

### 2. カレンダー描画位置のピクセル単位の調整
**【課題】**
カレンダービューにおいて、左側の時間ラベル（9:00など）と、実際の予定バー（Container）の開始位置が視覚的に数ピクセル〜数十ピクセルずれて見え、ユーザーに違和感を与える問題がありました。

**【解決策】**
グリッド線の描画開始位置と、時間ラベルのフォントサイズ・行間を計算し、`top` 位置を微調整（オフセット計算）しました。これにより、設定した時間と見た目の位置が完全に一致するUIを実現しました。

### 3. モードに応じた情報の「出し分け」
週表示（Week View）ではスペースが狭いため、文字を表示するとレイアウト崩れ（Overflow）が頻発しました。
そこで、**「週表示では色帯のみでシンプルに」「日表示では詳細テキストを表示」**というように、表示モード（`CalendarViewMode`）に応じてWidgetの構成を動的に切り替えるロジックを実装し、視認性と情報量のバランスを最適化しました。
